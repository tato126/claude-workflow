# Refactoring Skill

## Description
코드 품질을 개선하고 구조를 정리하는 스킬입니다.

## When to use
- 코드 구조 개선
- 중복 코드 제거
- 성능 최적화
- 가독성 향상

## Instructions
1. 현재 상태 분석
   - 문제점 파악
   - 개선 포인트 도출

2. 리팩토링 계획
   - 변경 범위 최소화
   - 단계별 진행
   - 기능 동작 유지 확인

3. 리팩토링 수행
   - 한 번에 하나씩
   - 각 단계 후 테스트
   - 점진적 개선

4. 검증
   - 기존 테스트 통과
   - 기능 동작 동일

## Techniques
- Extract Function/Method
- Rename Variable/Function
- Move to separate file
- Simplify conditionals
- Remove dead code

## Principles
- 동작 변경 없이 구조만 개선
- 작은 단위로 진행
- 각 단계 검증

## Output
- 리팩토링된 코드
- 변경 사항 요약
